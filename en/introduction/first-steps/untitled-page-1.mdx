### Step 2: Describe Your Task (That's It!)

#### How to Ask

Simply type what you need. Be specific.

**Good Requests:**

```
✅ "Research AI trends for 2026 and create a 10-slide 
presentation with data visualizations"

✅ "Analyze my sales data from last quarter and 
create a financial report with insights"

✅ "Create a landing page for my new product with 
pricing tiers, features, and testimonials"

✅ "Find YouTube videos about home workouts, 
analyze the top 5, and create a 30-day plan"

✅ "Draft professional emails to follow up on 
these 5 prospects with personalized messages"
```

**Vague Requests (avoid):**

```
❌ "Do something"
❌ "Help me"
❌ "Make a thing"
❌ "Whatever you think"
```

#### Pro Tips for Better Results

**Be Specific**

```
Instead of: "Create a presentation"
Say: "Create a 12-slide presentation on 
sustainable fashion trends with images, data, 
and citations from 5+ sources"
```

**Provide Context**

```
Instead of: "Analyze this data"
Say: "Analyze our Q4 sales data (attached) 
for trends by region, product category, 
and customer segment"
```

**Include Requirements**

```
Instead of: "Write a proposal"
Say: "Write a 5-page proposal for our 
SaaS pricing service. Include: executive 
summary, 3 benefits with ROI data, 
implementation timeline, 3 pricing tiers"
```

**Specify Format**

```
Instead of: "Give me results"
Say: "Deliver as: PDF report with executive 
summary, detailed analysis, and 3 appendices"
```
