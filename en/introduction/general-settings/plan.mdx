## **Important Notes About Limits**

### **Customizable in Settings:**

All numerical limits shown (parallel tasks, scheduled tasks, app tasks, custom agents, credits) are **configurable in NeuroAI settings**. This means:

1. **Parallel Tasks (10)**
   - Can be adjusted up or down based on your workflow needs
   - Navigate to Settings → Execution → Parallel Task Limit
2. **Scheduled Tasks (50)**
   - Increase if you need more automation
   - Settings → Automation → Scheduled Task Limit
3. **App Tasks (200)**
   - Adjust based on integration volume
   - Settings → Integrations → App Task Limit
4. **Custom Agents (100)**
   - Scale up for larger teams
   - Settings → Agents → Agent Limit
5. **Credits (14,999)**
   - Request credit limit increases for heavy usage
   - Settings → Billing → Credit Management

**Note:** Limit adjustments may require account verification or plan upgrades depending on usage requirements.
