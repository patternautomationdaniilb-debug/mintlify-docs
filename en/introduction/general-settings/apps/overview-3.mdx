## Troubleshooting Apps

### App Not Working?

**Check These:**

- Is the app installed? (Go to Settings → Apps)
- Do you have the right permissions? (Reconnect the app)
- Is your account valid? (Verify login credentials)
- Is the service available? (Check app status page)

**Solutions:**

1. Disconnect and reconnect the app
2. Grant additional permissions
3. Check app documentation
4. Contact app support

### Slow App Response?

**Possible Causes:**

- App service is slow
- Network connection issue
- Too much data to process
- High usage periods

**Solutions:**

- Try again later
- Check internet connection
- Make request more specific
- Use pagination for large datasets

## Managing Your Apps

### View Connected Apps

```
Settings → Apps → View Connected
- See all installed apps
- Check permissions granted
- View last accessed date
- Disconnect unused apps
```

### Remove an App

```
Settings → Apps → [App Name] → Disconnect
- Revokes all permissions
- App no longer appears in Neuro
- Data not affected
- Reconnect anytime
```

### Update Permissions

```
Settings → Apps → [App Name] → Manage Permissions
- Add new permissions
- Revoke unnecessary access
- Re-authenticate if needed
```

## Best Practices

### ✅ Do's

- Grant minimal permissions needed
- Regularly review connected apps
- Use descriptive names for tokens
- Update passwords on connected accounts
- Remove unused apps

### ❌ Don'ts

- Don't share app tokens
- Don't use shared account credentials
- Don't grant all permissions if unnecessary
- Don't ignore app security updates
- Don't leave unused apps connected
